What are CSS Animations?

CSS animations allow you to create smooth transitions between different states of an element. They provide a way to animate HTML elements without using JavaScript or Flash. CSS offers two main ways to create animations: transitions and keyframes.

CSS Transitions

CSS transitions provide a simple way to animate changes from one CSS property value to another over time. They're triggered by state changes such as hover, active, or focus events.

Basic Transition

.box {
  width: 100px;
  height: 100px;
  background-color: #4361ee;
  transition: transform 0.3s ease, background-color 0.3s ease;
}

.box:hover {
  transform: scale(1.1);
  background-color: #3a0ca3;
}

Transition Properties

/* The transition shorthand property combines: 
   transition-property, transition-duration, 
   transition-timing-function, and transition-delay */

.element {
  transition: all 0.5s ease 0.2s;
}

/* This is equivalent to: */
.element {
  transition-property: all;
  transition-duration: 0.5s;
  transition-timing-function: ease;
  transition-delay: 0.2s;

CSS Keyframes

CSS keyframes allow you to create more complex animations than transitions. They let you define multiple steps in an animation sequence, giving you more control over how the animation progresses.

Basic Keyframes Animation

@keyframes pulse {
  0% {
    transform: scale(1);
    opacity: 1;
  }
  50% {
    transform: scale(1.2);
    opacity: 0.7;
  }
  100% {
    transform: scale(1);
    opacity: 1;
  }
}

.pulse {
  width: 100px;
  height: 100px;
  background-color: #4361ee;
  border-radius: 50%;
  animation: pulse 1.5s ease-in-out infinite;
}
Pulse

Multiple Properties Animation

@keyframes rotate-scale {

Animation Delay and Direction

.dot {
  animation: jump 1s infinite;
}

.dot:nth-child(2) {
  animation-delay: 0.2s;
}

.dot:nth-child(3) {
  animation-delay: 0.4s;
}

@keyframes jump {
  0%, 100% {
    transform: translateY(0);
  }
  50% {
    transform: translateY(-15px);
  }
}

Animation Fill Mode

.forwards {
  animation-fill-mode: forwards; /* Keeps the final state */
}

.backwards {
  animation-fill-mode: backwards; /* Applies first keyframe before starting */
  animation-delay: 1s;
}

.both {
  animation-fill-mode: both; /* Combines forwards and backwards */
  animation-delay: 1s;
}

Animation Timing Functions

Timing functions control the pace of the animation. They determine how intermediate values are calculated between the start and end states of an animation.

Common Timing Functions

/* Built-in timing functions */
.element1 { transition-timing-function: linear; }
.element2 { transition-timing-function: ease; }
.element3 { transition-timing-function: ease-in; }
.element4 { transition-timing-function: ease-out; }
.element5 { transition-timing-function: ease-in-out; }

/* Custom cubic bezier function */
.element6 { transition-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55); }

Practical Animation Examples

These examples show how animations can be applied to common UI elements to improve the user experience.

Button Hover Effect

.animated-button {
  position: relative;
  overflow: hidden;
  z-index: 1;
  transition: color 0.3s;
}

.animated-button::before {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  width: 0;
  height: 100%;
  background-color: #3a0ca3;
  z-index: -1;
  transition: width 0.3s ease;
}

.animated-button:hover::before {
  width: 100%;
}

Card Hover Effect

.animated-card {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.animated-card:hover {
  transform: translateY(-10px);
  box-shadow: 0 12px 24px rgba(0, 0, 0, 0.15);
}

.card-img {
  transition: transform 0.5s ease;
}

.animated-card:hover .card-img {
  transform: scale(1.1);
}

Typing Effect

.typing-effect {
  width: 21ch;
  white-space: nowrap;
  overflow: hidden;
  border-right: 3px solid #4361ee;
  animation: typing 3s steps(21) infinite, 
             blink 0.5s step-end infinite alternate;
}

@keyframes typing {
  0% { width: 0; }
  50%, 100% { width: 21ch; }
}

@keyframes blink {
  50% { border-color: transparent; }
}
