Quick Wins

Simple techniques that make a big difference with minimal effort.

Tip 1: Create advanced box-shadow style

Medium

Create an advanced box-shadow style that adds depth and dimension to your elements. 


.square {
  width: 50px;
  height: 50px;
  margin: 50px;
  background-color: plum;
  box-shadow: 60px 0 0 green, 120px 0 0 red, 0 60px 0 violet, 60px 60px 0 lightblue, 120px 60px 0 lightsalmon, 0 120px 0
  palegreen, 60px 120px 0 firebrick, 120px 120px 0 khaki;
}
  
.circle {
  width: 80px;
  height: 80px;
  margin: 0 auto;
  border-radius: 50%;
  background-color: orange;
  box-shadow: 40px 0 0 red;
}

How it works

This technique uses multiple box-shadow properties to create a layered effect, giving the illusion of depth and realism. By adjusting the offsets, blur radius, and color, you can achieve various styles.

Tip 2: Effect for textarea

Medium

Create a visually appealing effect for textarea elements to enhance user experience. 


.input-style {
  width: 100%;
  padding: 20px;
  border-radius: 8px;
  color: #333;
  border: solid 1px #c0c1c7;
  transition: all 0.25s linear;
  resize: none;
}

.input-style::-webkit-input-placeholder {
  color: #c0c1c7;
}

.input-style:focus {
  box-shadow: 0 0 0 4px #ffdcdc;
  border-color: #e5544a;
}

.button-style {
  width: 100%;
  padding: 20px;
  border-radius: 8px;
  margin-top: 20px;
  margin-bottom: 25px;
  border: 0;
  color: #fff;
  font-weight: bold;
  font-size: 18px;
  cursor: pointer;
  background-color: #e55371;
  box-shadow: 0 5px 0 0 #92172e;
  transition: all 0.25s linear;
}

.button-style:hover {
  background-color: #e5c4c3;
  color: #e55371;
}

.button-style:active {
  box-shadow: 0 2px 0 0 #92172e;
  transform: translateY(3px);
}

How it works

This technique uses CSS transitions and pseudo-elements to create a visually appealing effect for textarea elements. By applying styles on focus, you can enhance the user experience and make the textarea more engaging.

Tip 4: Create ribbon with clip-path property

Medium

Create a ribbon effect with the clip-path property to add depth and style to your elements. 


.polygon {
  width: 100px;
  height: 200px;
  background-color: #f00f0f;
  border-radius: 5px;
  margin: 20px auto;
  clip-path: polygon(0 0, 100% 0, 100% 100%, 50% 75%, 0 100%);
  /* x y coordinates for the polygon */             
}

How it works

This technique uses the clip-path property to create a ribbon effect. By defining a polygon shape, you can achieve a stylish look that enhances the overall design.

Tip 5: Handling complex backgrounds with clip-path

Advanced

Create a complex background effect using the clip-path property to enhance your design. 


.background-container {
  margin: 0 auto;
  height: 600px;
  position: relative;
}

.background-container::before {
  content: "";
  width: 100%;
  height: 100%;
  position: absolute;
  top: 0;
  left: 0;
  background-image: linear-gradient(to right bottom, #07047e, #1a3bda);
  z-index: -1;
}

@supports (clip-path: polygon(0 0, 100% 0, 100% 70%, 40% 80%, 39% 85%, 0 100%)) {
  .background-container::before {
    clip-path: polygon(0 0, 100% 0, 100% 70%, 40% 80%, 39% 85%, 0 100%);
    -webkit-clip-path: polygon(0 0, 100% 0, 100% 70%, 40% 80%, 39% 85%, 0 100%);
  }             
}

How it works

This technique uses the clip-path property to create a complex background effect. By defining a polygon shape, you can achieve a stylish look that enhances the overall design.

Tip 7: Make image video responsive to scale

Easy

Create a responsive image or video element that scales with the viewport size. 


.responsive-media {
  position: relative;
  height: 0;
  padding-bottom: 56.25%;
}

.responsive-media>* {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  object-fit: cover;             
}

How it works

This technique uses the padding-bottom property to create a responsive container. By setting the padding-bottom to a percentage, you can maintain the aspect ratio of the image or video element.

Tip 1: How to make video responsive in 16 / 9 ratio

Advanced

Create a responsive video element that maintains a 16:9 aspect ratio across different screen sizes. 


.media {
  height: 0;
  position: relative;
  padding-bottom: 56.25%;
  padding-bottom: calc(100% / (var(--ratio)));
}

.media iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;             
}

How it works

This technique uses the padding-bottom property to create a responsive container. By setting the padding-bottom to a percentage, you can maintain the aspect ratio of the video element.

Tip 2: Split form layout with CSS Grid

Medium

Create a responsive form layout that adapts to different screen sizes using CSS Grid. 


.form-layout {
  max-width: 500px;
  margin: 50px auto;
}

.form-group {
  margin-bottom: 40px;
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(0, 1fr));
  grid-column-gap: 40px;
}

.form-input {
  padding: 10px;
  border: solid 1px #f00;
  outline: none;             
}

How it works

This technique uses CSS Grid to create a responsive form layout. By defining grid-template-columns and grid-column-gap, you can achieve a flexible layout that adapts to different screen sizes.

Tip 1: How to use variables efficiently in CSS

Medium

Use CSS variables to create a consistent design system and make your code more maintainable. 


.card {
  --card-image-height: 200px;
  --card-content-padding: 20px;
  --card-title-color: #1a1a1a;
  width: 300px;
  margin: 25px auto;
  border: solid 1px #ccc;
}

.card-image {
  height: var(--card-image-height);
  object-fit: cover;
  width: 100%;
}

.card-content {
  padding: var(--card-content-padding);
}

.card-title {
  color: var(--card-title-color);
}

@media screen and (max-width: 767px) {
  .card {
    --card-image-height: 150px;
    --card-content-padding: 15px;
    --card-title-color: red;
  }             
}

How it works

This technique uses CSS variables to define reusable values for properties. By using variables, you can easily update styles across your entire stylesheet without having to change each instance individually.
